Property Description Values Value Description
display Flexbox সক্রিয় করার জন্য ব্যবহৃত হয়। flex, inline-flex flex: একটি block-level container তৈরি করে।
inline-flex: একটি inline-level container তৈরি করে।
flex-direction Flex items কোন direction-এ থাকবে তা নির্ধারণ করে। row, row-reverse, column, column-reverse row: Items left-to-right।
row-reverse: Items right-to-left।
column: Items top-to-bottom।
column-reverse: Items bottom-to-top।
justify-content Main axis বরাবর flex items কিভাবে align হবে তা নির্ধারণ করে। flex-start, flex-end, center, space-between, space-around, space-evenly flex-start: Items left/top।
flex-end: Items right/bottom।
center: Items center।
space-between: Items মাঝে সমান গ্যাপ।
space-around: Items চারপাশে গ্যাপ।
align-items Cross axis বরাবর flex items কিভাবে align হবে তা নির্ধারণ করে। flex-start, flex-end, center, baseline, stretch flex-start: Top aligned।
flex-end: Bottom aligned।
center: Centered।
baseline: Baseline-aligned।
stretch: Items fit করে।
align-self নির্দিষ্ট একটি flex item কিভাবে cross axis বরাবর align হবে তা নির্ধারণ করে। Same as align-items Item-specific alignment সেট করার জন্য ব্যবহৃত হয়। Default auto, যা parent এর align-items inherit করে।
align-content Multiple rows কিভাবে cross axis বরাবর align হবে তা নির্ধারণ করে। flex-start, flex-end, center, space-between, space-around, space-evenly, stretch Similar to justify-content, তবে multiple rows এর জন্য।
flex-wrap Flex items এক লাইনে থাকবে নাকি multiple lines-এ wrap করবে তা নির্ধারণ করে। nowrap, wrap, wrap-reverse nowrap: All items এক লাইনে থাকবে।
wrap: Items multiple lines-এ move করবে।
wrap-reverse: Reverse order-এ wrap করবে।
order Flex items-এর sequencing নির্ধারণ করে। Integer values (e.g., 0, 1, -1) Smaller values আগে, বড় values পরে render হয়। Default value: 0।
flex Flex item এর grow, shrink এবং basis কিভাবে কাজ করবে তা নির্ধারণ করে। none, auto, [flex-grow] [flex-shrink] [flex-basis] none: No growth/shrinkage।
auto: Fully flexible।
Example: 1 1 auto।
flex-grow Flex item কতটুকু grow করবে তা নির্ধারণ করে। Numeric value (e.g., 0, 1, 2) 0: No growth।
Positive value: Proportional growth।
flex-shrink Flex item কতটুকু shrink করবে তা নির্ধারণ করে। Numeric value (e.g., 0, 1, 2) 0: No shrink।
Positive value: Proportional shrink।
flex-basis Flex item এর initial size নির্ধারণ করে। auto, length values (e.g., px, %, em) auto: Content size অনুযায়ী।
Length value: Fixed size।
gap Flex items এর মধ্যে gap নির্ধারণ করে। Length values (e.g., px, %, em) Items এর মধ্যে horizontal এবং vertical gap নির্ধারণ করে।
Property Description Values Value Description
display গ্রিড সক্রিয় করার জন্য ব্যবহৃত হয়। grid, inline-grid grid: Block-level গ্রিড কন্টেইনার।
inline-grid: Inline-level গ্রিড কন্টেইনার।
grid-template-rows গ্রিডের জন্য প্রতিটি সারির আকার নির্ধারণ করে। Length, Percentage, auto, repeat(), minmax() প্রতিটি সারির আকার যেমন 100px, 1fr, minmax(100px, auto) ইত্যাদি।
grid-template-columns গ্রিডের জন্য প্রতিটি কলামের আকার নির্ধারণ করে। Length, Percentage, auto, repeat(), minmax() প্রতিটি কলামের আকার যেমন 200px, 1fr, minmax(200px, auto) ইত্যাদি।
grid-template-areas গ্রিড লেআউটের জন্য নামকৃত এলাকা নির্ধারণ করে। String values representing grid areas উদাহরণ: "header header" "sidebar main" "footer footer"।
grid-row একটি আইটেম কোন গ্রিড লাইনে শুরু এবং শেষ হবে তা নির্ধারণ করে। Line number, span <number>, auto উদাহরণ: 1 / 3, যা গ্রিডের প্রথম লাইন থেকে শুরু করে তৃতীয় লাইনে শেষ হবে।
grid-column একটি আইটেম কোন কলামে শুরু এবং শেষ হবে তা নির্ধারণ করে। Line number, span <number>, auto উদাহরণ: 2 / 4, যা গ্রিডের দ্বিতীয় থেকে চতুর্থ কলামে থাকবে।
gap গ্রিড আইটেমগুলির মধ্যে ফাঁক নির্ধারণ করে। Length values (e.g., 10px, 1em, %) সারি এবং কলামের মধ্যে ফাঁক নির্ধারণ করে।
row-gap শুধুমাত্র সারির মধ্যে ফাঁক নির্ধারণ করে। Length values (e.g., 10px, 1em) সারির মধ্যে নির্দিষ্ট পরিমাণ ফাঁক।
column-gap শুধুমাত্র কলামের মধ্যে ফাঁক নির্ধারণ করে। Length values (e.g., 10px, 1em) কলামের মধ্যে নির্দিষ্ট পরিমাণ ফাঁক।
justify-items গ্রিড আইটেমগুলো main axis বরাবর কিভাবে align হবে তা নির্ধারণ করে। start, end, center, stretch উদাহরণ: center, যা center-align করে।
align-items গ্রিড আইটেমগুলো cross axis বরাবর কিভাবে align হবে তা নির্ধারণ করে। start, end, center, stretch উদাহরণ: stretch, যা items পুরো height নিয়ে নেবে।
place-items align-items এবং justify-items এর shorthand প্রপার্টি। Combination of align-items এবং justify-items উদাহরণ: center start, যা cross axis-এ center এবং main axis-এ start করে।
justify-content পুরো গ্রিড কন্টেইনার main axis বরাবর কিভাবে align হবে তা নির্ধারণ করে। start, end, center, space-between, space-around, space-evenly উদাহরণ: space-between, যা গ্রিড আইটেমের মাঝে সমান ফাঁক তৈরি করে।
align-content পুরো গ্রিড কন্টেইনার cross axis বরাবর কিভাবে align হবে তা নির্ধারণ করে। start, end, center, space-between, space-around, space-evenly, stretch উদাহরণ: stretch, যা পুরো height পূরণ করে।
place-content align-content এবং justify-content এর shorthand প্রপার্টি। Combination of align-content এবং justify-content উদাহরণ: center stretch, যা cross axis-এ center এবং main axis-এ stretch করে।
grid-auto-rows অটো-জেনারেটেড সারির আকার নির্ধারণ করে। Length, Percentage, auto, minmax() উদাহরণ: minmax(100px, auto)।
grid-auto-columns অটো-জেনারেটেড কলামের আকার নির্ধারণ করে। Length, Percentage, auto, minmax() উদাহরণ: 200px।
grid-auto-flow অটো-প্লেসমেন্ট এলগরিদমের জন্য নির্ধারণ করে। row, column, dense, row dense, column dense উদাহরণ: row dense, যা items ঘনভাবে প্লেস করে।
z-index আইটেম কোন লেয়ারে প্রদর্শিত হবে তা নির্ধারণ করে। Integer values (e.g., 1, 10, -1) Positive value উপরে, Negative value নিচে দেখায়।
